# PDF Manipulator (Python OOP Project)

## Overview
This project was developed to practice Object-Oriented Programming in Python.
It allows combining specific pages from multiple PDFs and applying a watermark.

## Features
- Extract selected pages from multiple PDFs
- Combine them into a new PDF
- Apply watermark to each page
- Basic error handling

## Technologies Used
- Python
- PyPDF2

## Learning Objectives
- Practice OOP design
- Work with external libraries
- Handle file operations
- Implement structured processing flow
